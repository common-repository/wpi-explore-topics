Lay all your tags/categories in alphabetical order on a page and search as you type (ajax) filter for a topic/tag
== Description ==

This plugin automatically lays all your tags/categories in alphabetical order on a page and adds an ajax filter input box which allows to search as you type for a particular topic/tag
It is a great way to show all the topics your site covers on various areas(keywords)

**Features**

1. Search for topics by tags or categoies as you type,
2. Supports Multisite,
3. Show your users the vast range of topics covered on your site,
4. Help in visitor retention and reduced Bounce Rate,
5. Simple and easy to configure

Here is a demo: 
http://appcoder.link/explore-topics/





== Installation ==

1. Upload the plugin files in the same way you'd install any other plugin.
2. Activate the plugin
3. Create a page and add shortcode [WPI_EXPLORE_TOPICS]
